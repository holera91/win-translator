# Windows Translation Script (Python)

Цей скрипт дозволяє швидко перекладати виділений текст за допомогою гарячих клавіш. 
Ідеально підходить для роботи через RDP (віддалений робочий стіл).

## Можливості
- **Ctrl+Shift+G**: Переклад на німецьку.
- **Ctrl+Shift+E**: Переклад на англійську.
- **Ctrl+Alt+Q**: Вихід із програми.

## Як встановити (для розробки)
1. Встановіть Python 3.x на Windows.
2. Відкрийте термінал (PowerShell/CMD) від імені Адміністратора.
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть скрипт: `python translator.py`

## Як зібрати в EXE (щоб працювало без Python)
Ви можете створити один файл, який буде працювати на будь-якому Windows:
```bash
pyinstaller --onefile --noconsole --uac-admin translator.py
```
Після цього у папці `dist/` з'явиться готовий файл `translator.exe`.
Його достатньо просто запустити від імені адміністратора.
