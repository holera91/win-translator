# Windows Translation Script (Python)

Цей скрипт дозволяє швидко перекладати виділений текст за допомогою гарячих клавіш. 
Ідеально підходить для роботи через RDP (віддалений робочий стіл).

## Можливості
- **Ctrl+Shift+G**: Переклад на німецьку.
- **Ctrl+Shift+E**: Переклад на англійську.
- **Ctrl+Shift+F**: Переклад на французьку.
- **Ctrl+Shift+S**: Переклад на іспанську.
- **Ctrl+Shift+I**: Переклад на італійську.
- **Ctrl+Alt+Q**: Вихід із програми.

## Як створення роботи

### Передумови
- Windows 10/11
- Python 3.7+
- Права адміністратора (для реєстрації гарячих клавіш)

### Встановлення
1. Клонуйте репозиторій або завантажте файли.
2. Відкрийте термінал (PowerShell/CMD) від імені Адміністратора.
3. Встановіть залежності:
```bash
pip install -r requirements.txt
```
4. Запустіть скрипт:
```bash
python translator.py
```

## Як використовувати

1. Виділіть текст, який хотіть перекласти.
2. Натисніть одну з гарячих клавіш (наприклад, **Ctrl+Shift+G** для німецької).
3. Текст автоматично:
   - Скопіюється до буфера обміну
   - Перекладеться
   - Замінить оригінальний текст

Для виходу натисніть **Ctrl+Alt+Q**.

## Як зібрати в EXE (щоб працювало без Python)

Ви можете створити один файл, який буде працювати на будь-якому Windows:
```bash
pyinstaller --onefile --noconsole --uac-admin translator.py
```
Після цього у папці `dist/` з'явиться готовий файл `translator.exe`.
Його достатньо просто запустити від імені адміністратора.

## Залежності

- **pynput** - для управління клавіатурою
- **pyperclip** - для роботи з буфером обміну
- **deep-translator** - для перекладу з Google Translate
- **pywin32** - для роботи з Windows API

## Деталі реалізації

Скрипт використовує:
- `win32gui.RegisterHotKey()` - для регістрації гарячих клавіш на рівні операційної системи
- `GoogleTranslator` - для перекладу
- Окреме вікно для обробки системних повідомлень
