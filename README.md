# Windows Translation Script (Python)

Цей скрипт дозволяє швидко перекладати виділений текст за допомогою гарячих клавіш. 
Він ідеально підходить для роботи через RDP (віддалений робочий стіл) та в повсякденних завданнях, оскільки автоматизує процес "Копіювати -> Перекласти -> Вставити".

## Можливості
- **Global Hotkeys**: Працює на рівні всієї системи через Windows API.
- **Автоматизація**: Самостійно копіює виділений текст, перекладає та замінює оригінал.
- **Трей (System Tray)**: Працює у фоновому режимі, керування через іконку біля годинника.
- **Гнучкі налаштування**: Можливість змінювати гарячі клавіші через графічне меню.
- **Google Translate**: Використовує бібліотеку `deep-translator` для доступу до Google Translate.

## Використані бібліотеки
- `pynput` - для симуляції натискань клавіш (Ctrl+C, Ctrl+V).
- `pyperclip` та `pywin32` - для надійної роботи з буфером обміну Windows.
- `deep-translator` - ядро перекладу (Google Translate API).
- `pystray` та `pillow` - для створення іконки в треї та меню.
- `tkinter` - для вікна налаштувань.
- `pyinstaller` - для збірки в EXE.

## Як встановити та запустити

### Передумови
- Windows 10/11
- Python 3.7+ (для запуску з коду)
- Права адміністратора (необхідні для реєстрації системних гарячих клавіш)

### Встановлення
1. Клонуйте репозиторій: `git clone https://github.com/holera91/win-translator.git`
2. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустіть скрипт:
   ```bash
   python translator.py
   ```

## Як зібрати в EXE (працює без Python)

Для створення автономного файлу використовуйте:
```bash
pyinstaller --onefile --noconsole --uac-admin translator.py
```
Готовий файл з'явиться у папці `dist/`. Прапор `--uac-admin` гарантує, що Windows запитає права адміністратора при запуску.

## Як використовувати

1. Виділіть будь-який текст.
2. Натисніть гарячу клавішу:
   - `Ctrl+Shift+G`: Переклад на німецьку.
   - `Ctrl+Shift+E`: Переклад на англійську.
   - `Ctrl+Shift+F`: Переклад на французьку.
   - `Ctrl+Shift+S`: Переклад на іспанську.
   - `Ctrl+Shift+I`: Переклад на італійську.
3. Оригінальний текст буде автоматично замінено на перекладений.

Для зміни клавіш натисніть правою кнопкою на іконку **T** у треї та виберіть **"Налаштування"**.
